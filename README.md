# ITC122-group3-school-website

ITC 122 - Intro to Web Design - Group 3 Project Website

## 🚀 Features

- 🏠 Modern homepage with hero section and school highlights
- 📊 Stats and school information display
- 📚 About page with school history, mission, and vision
- 🧭 Programs and offerings section
- 📱 Fully responsive design (mobile-friendly)
- ⚡ Fast routing with TanStack Router
- 🎨 Styled using Tailwind CSS

---

## 🛠️ Tech Stack

- **Frontend:** React + TypeScript
- **Build Tool:** Vite
- **Routing:** TanStack Router
- **Styling:** Tailwind CSS
- **Icons:** Lucide React

---

## 📂 Project Structure

⚙️ Installation

# Clone the repository

- git clone https://github.com/vincel08/ITC122-group3-school-website.git
- cd REPO_NAME

# Install dependencies

- npm install

# Start development server

- npm run dev

- Then open: http://localhost:5173

# Build for Production

- npm run build

# 👀 Preview Production Build

- npm run preview
